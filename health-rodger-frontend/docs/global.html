(constant) MainListItems

MainListItems renders navigation list items dynamically based on the user's role. It provides a set of navigation buttons that are conditionally rendered to match the access level of the user (Hospital, Researcher, Admin). Clicking on any of these buttons will change the currently active page in the main application dashboard.
